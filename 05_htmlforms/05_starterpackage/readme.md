# HTML5 forms
## Aufgabe

Erstellen Sie ein HTML Formular mit diesen Eingabefeldern:

~~~
  Suchbegriff
  Datum
  Ort
  Umkreis
  Button „Senden“
~~~

Nutzen Sie die soeben kennengelernten Funktionalitäten für ein Formular
